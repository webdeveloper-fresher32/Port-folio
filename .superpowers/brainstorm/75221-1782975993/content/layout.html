Which overall page layout works better?

Both use the dark/technical style. Difference is single-page scroll vs multi-page nav.

A

Single-page scroll

One long page: Hero → About → Skills → Experience → Projects → Contact, with a sticky nav that jumps to sections. Fast to browse, recruiter-friendly, easy to deploy.

B

Multi-page

Separate routes: Home, Projects (with individual project detail pages), Experience, Contact. More like a full site, better for deep case studies and SEO per project, more work to build.
